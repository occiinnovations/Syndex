# Syndex

A Chrome extension that shows you synonym suggestions while you write. Hold and select a word, scroll through options, and click to swap it in. Works on Gmail, most websites, and text inputs. This does NOT work universally however; no guarantees.

Syndex uses the Datamuse API to fetch up to 100 synonyms per word. No login required, no tracking.
