# Syndex

A Chrome extension that shows you synonym suggestions while you write. Hold and select a word, scroll through options, and click to swap it in. Works on Gmail, most websites, and text inputs. 
Syndex uses the Datamuse API to fetch up to 100 synonyms per word. No login required, no tracking. 

# How to Use

Double tap a word in Gmail on your chrome browser. A box underneath/nearby should pop up, and you may scroll on that box to see different synonym options. When you see the synonym you prefer, simply click that synonym in the box, and the word will be replaced in the email.

# Important notes

1. As of now, this project only works for Gmail.
2. This may not work perfectly in all cases. In order to debug, use inspect to verify if the system activates or not.
3. Syndex is an on-going project, more updates are planned.
